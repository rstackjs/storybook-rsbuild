# Framework

Storybook 支持众多前端技术栈——React、Vue、Web Components 等等。本节将介绍在使用 Storybook Rsbuild 时如何添加或扩展 framework 支持。

:::info
有关详细的贡献指南，请参阅 [Contributing a Storybook framework](https://storybook.js.org/docs/contribute/framework)。
:::
